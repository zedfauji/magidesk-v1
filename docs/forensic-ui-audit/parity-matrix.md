# UI Feature Parity Matrix

## Classification Legend

| Status | Description |
| :--- | :--- |
| **IMPLEMENTED** | The feature is fully implemented and matches forensic requirements. |
| **PARTIAL** | The feature is partially implemented, but key behaviors or UI elements are missing. |
| **NOT IMPLEMENTED** | The feature is missing entirely from the UI. |
