# Configuration Guide

Once logged in (default PIN: `1234` for Admin), you can configure the specific hardware and behavior for this terminal.

## Accessing Settings
1. Log in as a Manager.
2. Tap the **Menu** icon (top-left) or **Admin** button.
3. Select **Configuration**.

## Terminal Settings

### Printer Setup
Magidesk supports Windows printers.
- **Receipt Printer**: Select your thermal printer (e.g., Epson TM-T88, Star Micronics).
- **Kitchen Printer**: Select the printer located in the kitchen.
- **Print Copies**: Choose how many receipts to print by default (usually 1 or 2).

### Terminal Identity
Each computer has a unique ID.
- **Terminal Name**: Give this station a recognizable name (e.g., "Bar 1", "Patio 2").
- **Table Layout**: Select which floor plan this terminal should display by default.

## Restaurant Settings (Global)
*Changes here affect ALL terminals.*

- **Restaurant Name**: Appears on receipts and reports.
- **Address & Phone**: Printed on customer receipts.
- **Tax Rate**: Set your local sales tax percentage (e.g., `8.25`).
- **Currency Symbol**: `$` (default).

## System Backup
It is recommended to back up your configuration periodically.
- Go to **Admin > Maintenance**.
- Click **Backup Configuration**.
- Save the file to a safe location (like a USB drive).
