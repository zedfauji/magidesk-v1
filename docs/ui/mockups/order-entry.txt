KEYBOARD SHORTCUTS:
==================

F1: Help
F2: Search menu items
F3: Switch order type
F4: Guest count
F5: Refresh
F6: Send to kitchen
F7: Print ticket
F8: Payment
F9: Notes
F10: Settings
Ctrl+S: Save ticket
Ctrl+P: Print
Ctrl+N: New ticket
Ctrl+F: Find
Ctrl+Z: Undo last action
ESC: Cancel current operation
Enter: Add selected item
Delete: Remove selected item
Arrow keys: Navigate items
Tab: Navigate between panels

ACCESSIBILITY FEATURES:
======================

Screen Reader Support:
- All interactive elements have ARIA labels
- Live regions for dynamic content updates
- Proper heading structure (H1-H6)
- Table headers for ticket data
- Button states announced

Keyboard Navigation:
- Full keyboard access to all features
- Logical tab order
- Focus indicators visible
- Skip links for main content
- Keyboard shortcuts documented

High Contrast Mode:
- High contrast color scheme
- Increased text contrast ratios
- Clear focus indicators
- Enhanced button borders

Text Scaling:
- Support up to 200% text scaling
- Layout adapts to larger text
- No horizontal scrolling at 150% scaling
- Touch targets maintain minimum size

This mockup represents the primary order entry interface used by servers and staff to take customer orders. The design emphasizes efficiency with large touch targets, clear visual hierarchy, and comprehensive information display while maintaining a clean, uncluttered appearance.
